Trivia Game

A trivia game with a "Jeopary" theme that uses JavaScript and jQuery.

LIRI stands for Language Interpretation and Recognition Interface. LIRI is a CLI Node application that takes in parameters and makes API calls to get back data.

BAmazonCLI

"BAmazon" is an easy to use CLI-based storefront using a MySQL database for the backend.

Bananas-2-Bananas
